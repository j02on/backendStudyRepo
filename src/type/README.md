## 기본 타입과 참조 타입

### 기본 타입이란?
> 기본 타입은 값을 저장할 때 실제 값이 저장된다. 
> 기본 타입의 종류에는 정수형(byte, short, int, long), 실수형(double, float), 문자형(char), 논리형(boolean)

### 참조 타입이란?
> 값을 저장할 떄 데이터가 저장된 위치를 나타내는 주소값이 저장된다. 객체의 주소를 저장하며 기본형을 제외한 나머지 타입을 뜻한다.
> 참조 타입은 객체이다.
> null을 가질 수 있다.

### 리터럴이란?
- 리터럴은 프로그래밍에서 값 자체를 직접 코드에 작성한 것을 말한다.
- 변수에 할당되는 값 자체를 뜻한다.
```java
int num;
num = 2;
//num을 초기화한 값인 2가 리터럴값이다.
```

### 기본 타입과 참조 타입의 차이점
| 구분         | 기본 타입        | 참조 타입               |
| ---------- | ------------ | ------------------- |
| 값 저장       | 값 자체         | 객체 주소(reference) 저장 |
| 객체 여부      | ❌            | ✅                   |
| 메서드 사용     | ❌            | ✅                   |
| null 가능 여부 | ❌            | ✅                   |
| 예시         | `int i = 3;` | `String s = "Hi";`  |

### 기본 타입과 참조 타입의 예시

| 분류       | 타입           | 예시                         | 객체 여부 | null 가능 |
| ---------- | ------------- | ---------------------------- | -------- | -------- |
| 기본 타입  | byte          | `byte b = 1;`                | ❌       | ❌       |
| 기본 타입  | short         | `short s = 2;`               | ❌       | ❌       |
| 기본 타입  | int           | `int i = 3;`                 | ❌       | ❌       |
| 기본 타입  | long          | `long l = 4L;`               | ❌       | ❌       |
| 기본 타입  | float         | `float f = 1.0f;`            | ❌       | ❌       |
| 기본 타입  | double        | `double d = 1.0;`            | ❌       | ❌       |
| 기본 타입  | char          | `char c = 'A';`              | ❌       | ❌       |
| 기본 타입  | boolean       | `boolean flag = true;`       | ❌       | ❌       |
| 참조 타입  | 클래스         | `String s = "Hi";`           | ✅       | ✅       |
| 참조 타입  | 배열           | `int[] arr = new int[5];`    | ✅       | ✅       |
| 참조 타입  | 인터페이스      | `Runnable r = new Thread();` | ✅       | ✅       |
| 참조 타입  | 추상 클래스     | `Animal a = new Dog();`      | ✅       | ✅       |
| 참조 타입  | Enum          | `Day d = Day.MON;`           | ✅       | ✅       |
| 참조 타입  | Annotation    | `@Override`                  | ✅       | ✅       |
| 참조 타입  | Wrapper 클래스 | `Integer n = 10;`            | ✅       | ✅       |
| 참조 타입  | 제네릭 타입     | `List<String> list`          | ✅       | ✅       |
| 참조 타입  | 내부 클래스     | `Outer.Inner i`              | ✅       | ✅       |
| 참조 타입  | 익명 클래스     | `new Runnable(){}`           | ✅       | ✅       |
| 참조 타입  | 람다식        | `() -> {}`                   | ✅       | ✅       |
| 참조 타입  | Record        | `record User()`              | ✅       | ✅       |
